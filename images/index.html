    Images on the Web

    by Liam Doyle

    image design by © Liam Doyle

    Website images are one of several types of web content. Most images on any website can be catagorized into 5 diferent areas. These areas include Logos, Photos, Illustrations, Mascots, and 3D Renders.

    What can images do?

    • They can draw attention and trigger an emotional response.
    • They can help users to orientate
    • Background images create atmosphere

    Tips for using images

    • Make sure the images are relevant
    • Use original images
    • Don't use just photos
    • Think about cropping and resizing
    • Use multiple images for products
    • Integrate icons
    • Optimize your images
