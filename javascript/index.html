    javascript frameworks

    by your Maryn Moore

    image design by © Maryn Moore

    Javascript is a “language” where one scripts to create and control web developments such as slideshows, graphics, and forms that can be interacted with. Javascript frameworks are basically pre-coded things that you might need that you can take the code from and apply it to what you need to do.

    A couple of the big frameworks: ReactJs, Vue, jQuery.

    ReactJs: Its used to build websites with interactive components. These can be menus, search bars, buttons, etc.

    Vue: Is a progressive framework that is used to build user interfaces. It is designed to be extremely easy to pick up and able to be put into other projects that are already existing.

    jQuery: A fast form of the javascript frameworks library. It makes the HTML document(s) much easier to manipulate, handle, and use animation. It also works across many browsers.
