    in-browser coding tools

    by Lowell Halliwill

    image design by © Lowell Halliwill

    What are they?

    In-browser coding tools. Every modern computer has these powerful tools. Some of these powerful tools inspect HTML, javascript, and CSS. These tools allow you to do a variety of things to web developers' code, like debugging and testing it. These tools make it so you can see how other people produce their websites using code. Each browser is a little different in how it works but overall it's pretty interesting and once you understand what they do they can be a huge help.

    How they work

    To inspect any page's code by using browser developer tools simply right click and click inspect or inspect element. Once you click inspect it should bring up a small tab that shows you all the elements used to create and put together the website you are on. Along with the element panel, there is a style panel. This contains the CSS It also allows you to see the console, sources, network, performance, memory, application, and security. You can also pinpoint a certain element you may want to inspect by highlighting it first then right-click and click inspect. You can also put your code on the page. Of course, none of this will change the view of the page for others but it will temporarily change your view until you refresh the page.

    What they are used for

    In browser coding tools are used for a lot of things. They can be used to see a page's HTML or CSS. They can be used for help or inspiration on creating your HTML or CSS. It can be used to temporarily see how a page would look if you changed a line of code or switched up the CSS. They make no direct creations of a webpage nor do they edit them, but they are used for web applications and user interfaces.

    Differences by browser

    Each browser coding tool changes depending on which browser you decide on using. For example, Google comes with Chrome DevTools. In other browsers you may have to download dev tools on your own or choose between several different dev tools to download. An example of this would be Mozilla Firefox. With Firefox you get to choose between either getting Firebug or Firefox Developer Edition. For the most part, the coding tools are all pretty similar and most of them get the job done. It's never a terrible thing though to go and check out different browser coding tools to test and compare and see which one you like better.
