    web hosting, servers, and clients

    by Haley Hansen

    image design © Haley Hansen

    Have you ever wondered how the interweb works? Well I’m here to tell you about Web hosting, web servers and web clients. The internet has been around for more than 50 years ago, it started up because it was going to be used as a weapon for the Cold War. Plus they have made numerous updates to it to make sure users have the best experience. And in today’s world the internet is broken down into any different parts such as servers for one example.

    Web servers are computer software that uses hypertext transfer protocol (HTTP). Along with other protocol’s to assure clients are responded to for their client requests that are made over the world wide web. All computers that host websites must have web server programs on their device in order to access the websites. Some examples on how they are used is for sending emails, which everyone does nowadays, building/publishing web pages and downloading the requests for the FTP which stands for file transfer protocol. There's also Dynamic and Static web servers which means the dynamic content can be updated while static content is what’s being shown.

    The user side of the web is known as web client, it usually refers to the users device they are using for an example could be a computer or any mobile device used. It’s also a side component within J2EE (Java 2 platform enterprise edition) which is an application model for developing and building enterprise applications. You can access web clients by using browsers that are already installed on your device you're using for example, Safari, Firefox, Google Chrome, and many more. Also can be known for helper applications and extensions to enhance the browser for the special services of the selected site.

    Lastly web hosting is a form of an internet hosting service which allows users to make their websites accessible to the WWW (World Wide Web). Web hosts are companies that provide space that is either owned or leased so their clients can use the website. Along with providing internet connectability which is often in the data center. The creator of the website can use SSL (secure sockets layer) which allows them to make their site and data more secure. There are also many different types of web hosting which vary greatly like a shared web hosting, managed hosting and cloud hosting, but there are many others in this area.
